# DESIGN SYSTEM | V5 Partners

## Identidade Visual
- **Cores Primárias**:
  - `background`: #000000 (Black)
  - `foreground`: #FFFFFF (White)
  - `primary`: #FFFFFF (White contrast)
  - `muted`: #71717A (Zinc-500)
  - `border`: #27272A (Zinc-800)

- **Tipografia**:
  - **Display**: Plus Jakarta Sans (Headers)
  - **Body**: Inter (Textos corridos)
  - **Brand**: Elza (Custom font files in `public/fonts`)

- **Componentes**:
  - `radius`: 0.75rem (Padrão para cards e botões)
  - `shadows`: Soft glow on dark background for active states.

- **Espaçamento**:
  - Grid base de 4px (Utilizar classes Tailwind `p-4`, `m-2`, etc.)
