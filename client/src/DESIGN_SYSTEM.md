# Design System - V5 Partners

Este documento define o guia visual e os padrões de interface da plataforma V5 Partners.

## Paleta de Cores
- **Primária (Ação)**: `#000000` (Preto profundo para sofisticação)
- **Secundária**: Tons de Cinza (Escala Zinc do Tailwind)
- **Sucesso**: Esmeralda/Verde para indicadores positivos
- **Aviso**: Âmbar para alertas
- **Erro**: Destrutivo/Vermelho para ações críticas

## Tipografia
- **Títulos (Headings)**: `Plus Jakarta Sans`
 - Pesos: 600 (Semi-bold), 700 (Bold)
- **Corpo (Body)**: `Inter`
 - Pesos: 400 (Regular), 500 (Medium)

## Componentes Principais
- **Botões**: Arredondamento padrão `md` (8px), estados de hover com leve escala ou mudança de opacidade.
- **Cards**: Bordas suaves, sombras sutis (`shadow-sm` ou `shadow-md`), fundo branco ou levemente acinzentado.
- **Inputs**: Foco com anel de cor primária, labels claros e placeholders discretos.
- **Sidebar**: Design colapsável, ícones Lucide com traço de 1.5px.

## Espaçamento e Layout
- **Grit**: Sistema de 4px (padrão Tailwind).
- **Containers**: Largura máxima controlada para leitura confortável.
- **Padding**: Generoso para evitar poluição visual.

## Animações
- Transições de página suaves (Framer Motion ou CSS Transitions).
- Feedback visual imediato em interações (hover, active).
